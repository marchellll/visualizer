## Project setup
```sh
npm i -g pnpm # if you haven't installed pnpm
pnpm install
```

### Compiles and hot-reloads for development
```sh
pnpm run serve
```

### Compiles and minifies for production
```sh
pnpm run build
```

### Lints and fixes files
```sh
pnpm run lint
```

## Deploying to Vercel
JUST PUSH TO MASTER

vercel will pick up the changes and deploy it to the domain
